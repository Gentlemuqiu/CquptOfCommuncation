

## 重邮交流论坛 ##

- 注册、登录与修改密码：用户可以轻松注册账户，登录后可随时修改个人密码。
- 发布论坛帖子：用户可以在论坛中发布各类帖子，分享资讯、提问或参与讨论。
- 观看帖子视频：用户可以观看与帖子相关的视频内容，丰富信息的呈现形式。
- 轮播图最新消息推送：通过轮播图展示最新的校园新闻和重要通知，确保用户及时获取更新信息。
- 发布论坛社区公告：管理员拥有发布社区公告的权限，用于发布重要事项或活动通知。
- 发送长评论：论坛成员可以发布详细的长评论，表达自己对帖子的看法。
- 发送短评论并有跟评功能：支持发送简短评论，并且可以对他人评论进行跟评，提升讨论的层次与互动性。
- 收藏帖子和评论：用户可以收藏感兴趣的帖子和评论，方便日后查阅。
- 举报违规发言：用户可以举报不当言论，确保社区环境的健康。
- 关注其他论坛成员：用户可以关注其他成员，方便获取他们发布的内容及互动信息。
- 个人资料完善与修改：用户可随时修改自己的个人资料，展示个人信息和兴趣。
- 管理员删除违规发言与帖子：管理员有权删除违反社区规则的发言和帖子，维护平台秩序。
- 管理员禁止用户评论或封停用户：管理员可根据违规行为对用户进行禁言或封停账号的处理。
- 点赞与踩功能：用户可以对帖子进行点赞或踩的操作，表达对内容的喜好或不同意见。
